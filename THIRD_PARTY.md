# Third-party software

## Airwindohhs

This project consumes the MIT-licensed [`jinpavg/airwindohhs`](https://github.com/jinpavg/airwindohhs) maintained fork of [`isabelgk/airwindohhs`](https://github.com/isabelgk/airwindohhs). Each generated catalog records the resolved fork commit and the underlying `airwindows/airwindows` source commit. The fork carries general header-generation and standalone-compile improvements, not Vroom's Unreal-specific edits. Airwindohhs remains an external dependency; its license must accompany binary distributions.

## godot-cpp

The GDExtension bindings use MIT-licensed [`godotengine/godot-cpp`](https://github.com/godotengine/godot-cpp), pinned at commit `ba0edfed90512ec64aba51d4295a3e7e30112f86` and generated against the Godot 4.6 API.
